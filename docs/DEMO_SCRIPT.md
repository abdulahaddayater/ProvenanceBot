# Demo Video Script (60–90 seconds)

**[0:00–0:10] Hook**
"Imagine reading an AI summary and wondering: did the sources really say that? ProvenanceBot fixes that — every citation is cryptographically linked to what was actually fetched, and stored on the Stellar blockchain."

**[0:10–0:25] Ask a question**
Show the home page. Type a real research question. Click Connect Wallet (Freighter testnet). Hit Ask ProvenanceBot.

**[0:25–0:40] Pipeline progress**
Point to the progress steps: Fetching sources → Summarizing → Anchoring on-chain.

**[0:40–0:55] Summary + chips**
Summary appears with numbered citation chips inline. Click chip [1]. Side panel / bottom sheet shows source title, URL, retrieval time, content hash.

**[0:55–1:10] On-chain verification**
Click "Verify on-chain." Green "Verified ✓" badge appears. Mention: "Anyone can check this on Stellar — no crypto expertise needed."

**[1:10–1:20] Mobile**
Quick cut to phone width — same flow, bottom-sheet modal.

**[1:20–1:30] Close**
"ProvenanceBot — verifiable research answers. Built on Stellar Soroban. Try the pilot at [URL]."
